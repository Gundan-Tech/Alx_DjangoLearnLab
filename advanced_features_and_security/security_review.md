Setting,Security Contribution
SECURE_SSL_REDIRECT,Eliminates plain-text communication by forcing encrypted connections.
HSTS Settings,Hardens the browser connection against protocol downgrade attacks (SSL Stripping).
SECURE_COOKIES,Prevents session hijacking by ensuring tokens never travel over unencrypted channels.
X_FRAME_OPTIONS,mitigates Clickjacking by preventing the UI from being embedded in malicious sites.
